CSS Takeaways

  • The browser is responsive. Use it to avoid Media Queries
  • As soon as you start setting values you might create issues
  • The body is the viewport. If you create a parent div it has 100% of the viewport
  • Try to use percentages for a width, not fixed pixels
  • The child container always get its percentages from the parent
  • In general try to avoid using height
  • Use em and rem that are relative measures. Usually 1 em is 16px. In CSS, "em" is a relative unit that scales based on the font size of the parent element, while "rem" scales based on the font size of the root element (usually the HTML tag), meaning "em" is relative to the immediate parent, while "rem" is relative to the entire document's root font size, providing more consistent scaling across the page.
